Kendaraan **"Only One"** adalah aset kendaraan kustom eksklusif yang dirancang khusus untuk satu pemilik di kota Bumantara Roleplay.

Untuk memastikan kualitas, optimisasi, dan keseimbangan server, setiap pengajuan aset kendaraan "Only One" **wajib** melalui proses peninjauan (vetting) oleh tim kami **sebelum** donasi dapat dilakukan.

### ğŸ¨ Kriteria & Penyesuaian
- **Kriteria Model:**
  - Desain harus realistis dan selaras dengan tema kota (*lore-friendly*).
  - Tidak mengandung desain atau *livery* yang berlebihan.
  - Tidak menggunakan efek visual ekstrem (misal: neon berlebihan, partikel).
- **Kriteria Handling (File Bawaan):**
  - Donatur **wajib menyertakan file `handling.meta`** bawaan dari aset tersebut.
  - Tim Manajemen berhak penuh untuk **menyesuaikan atau mengganti total** file handling tersebut agar seimbang dengan standar server.
- **Evaluasi Model:** Tim Manajemen berhak penuh untuk menolak atau meminta revisi jika model kendaraan tidak memenuhi standar teknis di atas.

### ğŸ”„ Prosedur Pengajuan dan Implementasi
Alur untuk menambahkan kendaraan "Only One" adalah sebagai berikut:

#### **Langkah 1: Pengajuan & Peninjauan (Vetting)**
- **Pengajuan File:** Donatur mengajukan file kendaraan (lengkap dengan `handling.meta`) kepada Tim Manajemen melalui tiket donasi.
- **Proses Peninjauan:** Tim akan memeriksa file untuk memastikan kesesuaian dengan semua Persyaratan Teknis dan Kriteria Model.
- **Jika Ditolak:** Donatur akan diinformasikan mengenai masalahnya (misal: poly count terlalu tinggi) dan **wajib menyesuaikan filenya** terlebih dahulu.
- **Jika Diterima:** Tim Manajemen akan memberikan persetujuan bahwa file kendaraan **layak** untuk diproses lebih lanjut.

#### **Langkah 2: Pembayaran Donasi**
- **HANYA SETELAH** file kendaraan Anda dinyatakan **"Diterima"** oleh Tim Manajemen, Anda dapat melanjutkan ke tahap pembayaran donasi.
- **PENTING:** Jangan melakukan pembayaran *sebelum* file Anda mendapatkan persetujuan.

#### **Langkah 3: Request Kecepatan & Implementasi**
- **Request Kecepatan:** Setelah donasi dikonfirmasi, donatur dapat mengajukan **permintaan (request) kecepatan maksimal** (sebelum *max upgrade*).
- **Penyesuaian Tim:** Tim akan mengatur *handling* dan kecepatan kendaraan sesuai permintaan, dengan batas atas **tidak melebihi 270 km/jam** (batas kelas X).
- **Implementasi:** Kendaraan akan dijadwalkan untuk dimasukkan ke dalam server pada *restart* berikutnya.

#### **Langkah 4: Kesempatan Update**
- Setelah kendaraan berhasil diterima dan dimasukkan ke dalam server, Anda akan mendapatkan **2 (dua) kali kesempatan** untuk melakukan *update* pada file *model* kendaraan tersebut di masa mendatang (misal: perbaikan visual atau penyesuaian livery).
