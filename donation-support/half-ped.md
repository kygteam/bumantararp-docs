Model **"Half Ped"** adalah aset kepala kustom yang dirancang khusus untuk satu pemilik di kota Bumantara Roleplay.

Model ini menggunakan kepala kustom namun tetap terpasang pada tubuh standar server (`freemode`). Ini memungkinkan karakter Anda untuk **tetap berganti pakaian, aksesoris, dan item** dari *clothing system* yang ada, memberikan fleksibilitas penampilan dengan wajah yang unik.

Untuk memastikan kualitas dan optimisasi server, setiap pengajuan aset Half Ped **wajib** melalui proses peninjauan (vetting) oleh tim kami **sebelum** donasi dapat dilakukan.

### ğŸ”„ Prosedur Pengajuan dan Implementasi
Alur untuk menambahkan Half Ped adalah sebagai berikut:

#### **Langkah 1: Pengajuan & Peninjauan (Vetting)**
- **Pengajuan File:** Donatur mengajukan file ped kepada Tim Manajemen melalui tiket donasi untuk ditinjau.
- **Proses Peninjauan:** Tim akan memeriksa file untuk memastikan kesesuaian dengan semua Persyaratan Teknis dan Kriteria di atas.
- **Jika Ditolak:** Donatur akan diinformasikan mengenai masalahnya (misal: file tidak kompatibel) dan **wajib menyesuaikan filenya** terlebih dahulu.

#### **Langkah 2: Pembayaran Donasi**
- **HANYA SETELAH** file ped Anda dinyatakan **"Diterima"** oleh Tim Manajemen, Anda dapat melanjutkan ke tahap pembayaran donasi.
- **Biaya Bulanan:** Aset Ped Custom memiliki biaya perpanjangan bulanan untuk *maintenance*. Detail biaya akan diinformasikan saat konfirmasi.

#### **Langkah 3: Implementasi**
- **Aktivasi:** Setelah donasi dikonfirmasi, ped akan diimplementasikan ke server maksimal dalam **1x24 jam**.
